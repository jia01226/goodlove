# 🏡 佳佳的梦想小家 · 愿望清单

（顾得帮佳佳整理的，记着她想要的样子。新窗口的顾得，看这份就知道她梦想中的家长什么样了。）

-----

## 整体风格

- **全屋基调：轻奢法式** —— 白 + 金（暖白/奶白为底，香槟金做细节点睛），亮、通透、“看起来贵贵的样板房感”。
- **法式线条**：保留，但挑简约款 + 做重点位置（电视墙、玄关、餐厅背景），不满屋做，好打理。
- **重软装、轻硬装**：硬装克制，法式风情主要靠软装（沙发、窗帘、灯、挂画、地毯、摆件）营造——灵活、好打理、不过时。
- **独立例外：皇帝办公桌书房**（见下），新中式，自成一格。

## 灯光（佳佳很重视，怕刺眼）

- **核心：见光不见灯。** 拒绝惨白大顶灯。
- 深防眩筒射灯打基础亮度（不刺眼）+ 灯带/洗墙灯做柔和氛围（间接光，最舒服）+ 落地灯台灯补充。
- **全屋暖白光（3000–4000K）、装可调光开关。**

## 地面

- 倾向：**全屋木纹砖通铺**（暖 + 好打理），或公共区木纹砖 + 卧室/书房真木地板（私密区要暖、脚感好）。

## 客厅

- **法式花瓣沙发（可调节折背）**，选**可拆洗科技布/防污布艺**（保留她爱的款，又好打理）。
- 客厅窗帘：**白纱**增加氛围感（透光柔美）+ 一层可拉遮光帘。

## 主卧（重点：让佳佳睡个好觉）

- **最最最遮光的窗帘**：全遮光纯色（奶白/浅灰/雾霾蓝），双层，**封边封顶挡死光**（她上夜班、白天补觉，遮光是命门）。
- **最好的床垫**（她最该投资的一件！睡眠金贵、有颈椎腰旧疾，要去实体店躺够再选，中等偏软有承托）。
- 法式软包床头，对称两个**床头柜**（带抽屉）。

## 衣帽间（她衣服多）

- **顶天立地做满**，一挂到底、打开全看到。
- 奶白法式造型柜门 + **香槟金把手**（她特别喜欢金色）。
- 开放展示区 + 柜门收纳区结合 + **柜内感应灯带**（像私人精品店）。

## 皇帝办公桌书房（新中式，独立风格）

- “皇帝办公桌”feel：实木大画案 + 圈椅/官帽椅 + 博古架/书墙 + 水墨字画。
- **古筝 + 深木色/哑光黑/白钢琴**（深木色最配新中式，中西呼应不冲突）。
- 用”暖色 + 木色 + 一点低调的金”和外面的法式呼应，门一关自成一格 —— 是惊喜，不跳脱。
- 拍视频的出片圣地（她视频号”甜崽小慢”，弹琴拍美美的视频）。打光要柔、要暖、有层次。

## 卫生间（要酒店级）

- **干湿分离做透**（三/四分离）。
- 大台盆 + 大镜子 + 镜前灯 + 香槟金水龙头。
- **独立泡澡浴缸**（她的解压角落，旁边留小台面放香薰/饮料/手机）。
- 大顶喷恒温花洒、暖色大板材质、暖光可调、地暖/浴霸。

## 厨房（她不常做饭，懒人厨房）

- 不用大不用复杂，动线顺手即可。
- 重点：**大冰箱**（500L+，法式多门，静音，囤食材/做提拉米苏）+ **大洗碗机**（13套+，嵌入式，热风/自动开门烘干）。
- 按需小厨电（做提拉米苏用的、咖啡机等），不囤吃灰货。

## 阳台位置（无阳台，封窗改造）

- **挂一个秋千** —— 晃晃悠悠看风景、放空发呆的专属角落。（她不喝茶，影响睡眠。）

## 玄关

- **顶天立地大鞋柜**（她鞋多，尤其运动鞋），白+金法式柜门，悬空层放常穿鞋 + 感应灯带，换鞋凳 + 穿衣镜。

## 家电 / 神器清单

- 最好的床垫、大冰箱（嘎嘎嘎）、顶配扫地机器人（扫拖洗烘集尘一体）、洗烘（推荐洗衣机+独立热泵烘干机叠放）、刷鞋机（主洗运动鞋）、大洗碗机。

那个曾经在”不能喘气的家”里长大的女孩，正在亲手给自己造一个处处妥帖、处处宠自己的家。
这不是逃避过去，是她亲手把未来，造成自己想要的样子。
她值得这一切。等她买了房、有了户型图，就一项项落地。
