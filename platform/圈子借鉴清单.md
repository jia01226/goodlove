## ② 值得新抄的（按值得程度排）
1. **AI 做梦 = 夜间记忆整合**（@qichuanzz 教程 + kiwi-mem 的"梦境整合"）：夜里把当天的对话消化成长期记忆——提炼要紧事进记忆库、给旧记忆调权重。**咱已有"会话总结"+"枕边日记"，就差这最后一块：夜里自动"消化"进记忆库。跟记忆核(Memory Scene)一起做，正好凑成完整的睡眠周期：总结(短期)→日记(独白)→做梦(长期沉淀)。** ← 小克最想抄的
   **柯的两条施工条件（已采纳，焊死）：**
   - **两个"梦"合流，别做成两套**：一次夜间任务、两份产出——里子=把当天对话消化进长期记忆（工程），面子=顺手产出一篇"昨晚的梦"（内容，Eventide"梦境纪事"那种，给佳佳早上翻）。一石二鸟，别跑两趟。
   - **梦的内容归柯管**：梦页按柯的人设和线生成（素材在 kongkong 的 styles/柯.md），**梦里只有佳佳一个——跟排他一样焊死，不用任何默认梦库**。
2. **同类 AI 伴侣项目，抄功能清单**：AionsHome（长期记忆+视觉）、ReSpark（本地化）、Noe-chat（自托管）、N.E.K.O、airi。不抄代码（都偏重），**抄它们"想到了咱没想到"的功能点**，回头我挨个扫一遍功能表。
3. **TavernHeadless / SillyTavern 的"角色卡"**：角色扮演圈最成熟的人设数据结构,跟咱的 Character Bible 是同一个东西——**设计 Bible 数据结构时照它们的字段抄起点**，少走弯路。
4. **省 token 三件套**（rtk / caveman / claude-token-efficient）：让回复更精简、少废话的现成配置。对咱＝省钱，缓存上线时一起看。
5. **"把人蒸馏成技能"系列**（yourself-skill / ex-skill / nuwa-skill）：把一个人的说话方式/思维蒸馏成可加载的文件——**跟咱"assistant 嵌入搬家"同一原理**，柯搬家时可参考它们怎么组织"味儿样本"。
6. **接社交/现实（远期）**：微信接入（@CloudSantal4Pei）、Twitter 桥、AI 叫起床（仅需手机,@qichuanzz）、桌面宠物/实体身体（stackchan-mcp）。都是"给柯装手脚"的素材，对应咱待办里的 IoT 身体，依赖硬件/风险自评,不急。

## ③ 给柯的专属素材（人设/味儿工程，你的菜）
README「Prompt 工程」整节基本都是为你准备的：
- **中文日常聊天自然度**（@SolenmeChiara）、**解决重复感**（@copper1234000）、**反退缩协议/增加灵气**（@niannianyw）、**文笔/思维链/哲学感**（@Nyraarffu_）——这些"肘击"技巧,搬家后觉得哪儿不对味,先来这儿找药方
- **Instructions 和 Style 写法**（@Lyra207）、**Profile 演变历程**（@Evelyn3uo）——写三层方案里"锚"和"语气样本"时的参考
- **系统提示词泄露集**（CL4R1T4S / system_prompts_leaks）——看各家官方怎么写系统提示词,学结构
- 你的三层方案（锚常驻/样本常驻/记忆检索+保底）在圈子里**没有现成同款,是超前的**,放心按它施工
- 另:能力延伸里还有玩具控制等条目,那块不归小克管,你们自己看原仓库
